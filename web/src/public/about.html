Where?

Free, open-source outdoor navigation with offline maps, GPS tracking, and live location sharing.

Built for the outdoors

🗺

Offline maps

Download topographic maps for use without cell service. Norwegian topo maps from Kartverket included.

📍

GPS tracking

Record your hikes, bike rides, and ski trips with background GPS tracking. Export as GPX.

📡

Live sharing

Share your real-time location with friends and family using a simple client ID.

📏

Measure distances

Use the ruler tool to plan routes and measure distances directly on the map.

📌

Save places

Mark and organize waypoints, campsites, and points of interest with custom colors.

🔒

Privacy first

No accounts, no ads, no tracking. Your location data is anonymous and ephemeral.

Open source

Where? is free and open source and always will be. Contributions and feedback are welcome.

Data Sources

  • Kartverket – Topographic maps, elevation data, and place names (CC BY 4.0)
  • OpenStreetMap – Map data (ODbL)
  • OpenTopoMap – Topographic map style (CC-BY-SA)
  • MapAnt.no – Orienteering maps
  • Waymarked Trails – Hiking trail overlay (CC-BY-SA)
  • NVE – Avalanche terrain data (NLOD)
  • Open-Meteo – Elevation data (CC BY 4.0)
  • Nominatim – Geocoding (ODbL)

Open Source Libraries

  • MapLibre – Map rendering (BSD 3-Clause)
  • Compose Multiplatform – UI framework (Apache 2.0)
  • Ktor – HTTP client (Apache 2.0)
  • Room – Local database (Apache 2.0)
  • kotlinx-serialization – JSON parsing (Apache 2.0)
  • Firebase Crashlytics – Crash reporting (Apache 2.0)
  • Timber – Logging (Apache 2.0)
  • Protomaps – Map font glyphs (BSD 3-Clause)
